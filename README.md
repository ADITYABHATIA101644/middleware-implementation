# Middleware Implementation (Logging & Auth) 🚀
**Course Outcome:** CO3 & CO4 | **Experiment:** 2.2.1

## 📌 Overview
A Node.js/Express API that demonstrates custom middleware for request logging and JWT-based authentication, integrated with MongoDB Atlas.

## 🛠️ Features
- **Logging Middleware**: Captures method, URL, and response time.
- **Auth Middleware**: Protects routes using JSON Web Tokens.
- **Database**: Connected to MongoDB for user persistence.
- **Error Handling**: Centralized async error management.

## 🚀 Live Demo
https://middleware-implementation-kjvj.onrender.com/

## 📂 Structure
- `/middleware`: Custom logic for logs and security.
- `/routes`: API endpoints for Auth and Protected data.
- `/models`: Mongoose User schema.
